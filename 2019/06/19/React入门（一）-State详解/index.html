一、 Demo创建/下载

两种方法创建新的React APP

1. github下载

  • 下载git地址.
    • 解压-打开
    • npm install
    • npm start
    • y（3000端口任务在运行是否愿意运行在别的端口上？）

2. 自己的命令行创建

  • 找到自己心仪的文件夹 （全英文）
    • 用命令行打开并抵达心仪文件夹
    • npx create-react-app my-app
    • cd my-app
    • npm start

      注意
      npx在第一行不是一个错字 - 它是一个包转发工具，附带npm 5.2+。
      创建的时候已经默认运行过（npm install ）可以直接start
